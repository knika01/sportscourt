# sportscourt
EECS 497 Sports Court App

Create and join sports games

Foster new connections through sports

TO RUN:

`cd frontend`

`npx expo start`

OR QR CODE VIA Expo Go

Technologies: 
- React Native w/ TypeScript (Frontend)
- Node.js (Backend)
- Express.js (Backend)
- PostgreSQL (NeonDB)
- Railway (Backend Deployment)
- Expo Go (App Deployment)
